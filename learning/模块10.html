  • 模块10：过渡、变形与动画

模块概述

学习目标

  • 知识目标：掌握CSS变形（Transform）的常用函数；理解CSS过渡（Transition）的原理；掌握CSS动画（Animation）的关键帧技术。
  • 能力目标：能使用过渡实现属性变化的平滑动画；能对元素进行移动、旋转、缩放和倾斜；能创建复杂的自定义动画序列。
  • 素养目标：通过动画效果改善用户体验，激发探索复杂交互效果的学习内驱力。

模块时长

6学时

核心模块

在网页设计中，动画效果是提升用户体验和界面吸引力的重要手段。通过使用CSS过渡、变形和动画，我们可以创建流畅的交互效果和生动的视觉体验。本项目将系统讲解CSS动画的核心技术，使你能够设计出更具吸引力和专业感的交互界面。

任务列表

  • 任务10.1：使用CSS变形改变元素状态

    基础

    【任务描述】 打开配套资源中的素材任务10.1.html。素材文件提供了一个基础的HTML结构，包含了一些示例卡片，但没有添加具体的变形效果。请通过应用CSS变形，实现元素的平移、旋转、缩放、倾斜等2D变形以及3D变形效果，增强网页的交互性和视觉吸引力。

    📘 知识链接：CSS变形核心

    • transform属性：设置2D/3D变换，可叠加多个函数，顺序影响效果。默认值none。
    • transform-origin：定义变形基准点（默认中心）。支持left,center,right,top,bottom,百分比,像素。
    • transform-style：flat（2D）或preserve-3d（3D），用于创建3D空间。
    • backface-visibility：设置背面是否可见（visible/hidden），用于3D翻转。
    • perspective：透视距离，值越小透视越强。
    • 常用变形函数：translate()、rotate()、scale()、skew()、perspective()以及对应的3D函数（translate3d, rotate3d, scale3d等）。

  • 任务10.2：使用过渡效果实现平滑交互

    核心

    【任务描述】 现已创建一个包含多个交互元素的页面，素材任务10.2（素材）.html。实现了包括颜色、大小、位置、按钮、卡片和图片的悬停状态。本任务要求基于提供的素材文件，为网页元素添加平滑过渡的交互效果，使交互更加自然流畅。利用CSS的transition属性可以优化过渡的持续时间、缓动函数和触发条件。实现多种平滑交互效果：颜色过渡、大小过渡、位置过渡、按钮过渡、卡片过渡、图片过渡。

    📘 知识链接：过渡核心属性

    • transition-property：指定过渡属性（all | none | 属性名）。
    • transition-duration：持续时间（s/ms）。
    • transition-timing-function：缓动曲线（linear, ease, ease-in, ease-out, ease-in-out, cubic-bezier）。
    • transition-delay：延迟时间。
    • 简写：transition: property duration timing-function delay; 多组用逗号分隔。

  • 任务10.3：使用关键帧动画打造高级动画效果

    进阶

    【任务描述】 现已创建一个包含多个交互元素的页面，素材任务10.3.html。请使用CSS关键帧动画的元素，实现复杂的自定义动画序列，包括循环动画、交替动画和基于状态变化的动画。关键帧动画可以创建比过渡更复杂的动画效果。常用的效果设置包括：动画名称、动画持续时间、延迟、播放次数、方向、速度曲线等。

    📘 知识链接：关键帧动画核心

    • @keyframes 定义动画序列，可使用百分比或from/to。
    • animation-name / animation-duration 必需。
    • animation-timing-function：缓动曲线。
    • animation-delay：延迟，负值可跳过前N秒。
    • animation-iteration-count：次数或infinite。
    • animation-direction：normal, reverse, alternate, alternate-reverse。
    • animation-fill-mode：forwards/backwards/both 保持结束/开始状态。
    • animation-play-state：running/paused。
    • 简写：animation: name duration timing-function delay count direction fill-mode play-state;

    任务10.3 实施代码 (关键帧动画)
    /* 1.颜色变化动画 */
@keyframes colorChange {
    0% { background-color: var(--secondary); }
    50% { background-color: var(--accent); }
    100% { background-color: var(--success); }
}
#colorAnimationBox { animation: colorChange 3s infinite alternate; }

/* 2.移动动画 */
@keyframes move {
    0% { transform: translateX(0); }
    100% { transform: translateX(100px); }
}
#moveAnimationBox { animation: move 2s infinite alternate; }

/* 3.缩放动画 */
@keyframes scale {
    0% { transform: scale(1); }
    50% { transform: scale(1.5); }
    100% { transform: scale(1); }
}
#scaleAnimationBox { animation: scale 2s infinite; }

/* 4.旋转动画 */
@keyframes rotate {
    from { transform: rotate(0deg); }
    to { transform: rotate(360deg); }
}
#rotateAnimationBox { animation: rotate 3s linear infinite; }

/* 5.多重动画 */
@keyframes multi {
    0% { transform: translateX(0) rotate(0deg); background-color: var(--secondary); }
    50% { transform: translateX(50px) rotate(180deg); background-color: var(--accent); }
    100% { transform: translateX(0) rotate(360deg); background-color: var(--success); }
}
#multiAnimationBox { animation: multi 4s ease-in-out infinite; }

/* 6.弹跳动画 */
@keyframes bounce {
    0%, 100% { transform: translateY(0); }
    50% { transform: translateY(-30px); }
}
#bounceAnimationBox { animation: bounce 1s ease-in-out infinite; }

/* 7.旋转加载器 */
@keyframes spin {
    0% { transform: rotate(0deg); }
    100% { transform: rotate(360deg); }
}
#spinnerLoader { border: 5px solid #f3f3f3; border-top: 5px solid var(--secondary); animation: spin 1s linear infinite; }

/* 8.脉冲加载器 */
@keyframes pulse {
    0% { transform: scale(0.8); opacity: 0.7; }
    50% { transform: scale(1.2); opacity: 1; }
    100% { transform: scale(0.8); opacity: 0.7; }
}
#pulseLoader { background-color: var(--secondary); animation: pulse 1.5s ease-in-out infinite; }

/* 9.进度条动画 */
@keyframes progress {
    0% { transform: translateX(-100%); }
    100% { transform: translateX(400%); }
}
#animatedProgressBar { animation: progress 2s linear infinite; width: 25%; }

/* 10.悬停触发动画 */
@keyframes hoverEffect {
    0% { transform: scale(1) rotate(0deg); background-color: var(--secondary); }
    100% { transform: scale(1.2) rotate(10deg); background-color: var(--accent); }
}
#hoverAnimationBox:hover { animation: hoverEffect 0.5s forwards; }

/* 11.按钮点击动画 */
@keyframes clickAnimation {
    0% { transform: scale(1); background-color: var(--secondary); }
    50% { transform: scale(0.9); background-color: var(--accent); }
    100% { transform: scale(1); background-color: var(--secondary); }
}
#clickAnimationBtn:active { animation: clickAnimation 0.3s ease; }

总结与回顾

本项目围绕 CSS 动效三大核心技术展开教学，构建了过渡（状态触发平滑过渡）、变形（2D/3D 空间变换）、动画（多关键帧自定义序列）的完整知识框架；掌握了属性配置、函数应用、关键帧定义等实操技巧，能独立实现平移、旋转、缩放、循环动画等基础效果；具备了 "按需择术" 的综合应用能力 —— 简单交互用过渡、空间变换用变形、复杂序列用动画，可设计开发富有吸引力的现代化交互界面。
