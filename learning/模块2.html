模块概述

学习目标

  • 知识目标：掌握HTML文档基本格式与结构规范；熟练运用页面格式化标签组织网页内容；掌握图像标签的用法及图片路径配置；熟练使用无序列表、有序列表与定义列表；掌握超链接标签的使用及页面跳转实现；了解HTML5新增语义化标签的功能与应用场景
  • 能力目标：能够独立书写规范的HTML文档结构；能够合理运用各类标签定义网页元素；学会制作图文混排的页面效果；能够设计常见的网页列表模块；能够使用超链接实现页面间导航与跳转；能够使用语义化标签进行网页布局
  • 素养目标：培养规范的代码编写习惯，遵循语法规则与行业规范，确保代码的可读性、可维护性；建立知识产权保护意识，在使用图像、字体等外部资源丰富网页基本元素时，规范引用，主动获取授权，杜绝侵权行为

建议学时

18学时

基础模块

在互联网时代，网页是信息传播与交互的核心载体，而网页的基本元素则是构建网页的"基石"。无论是日常浏览的新闻网站、购物平台，还是使用的各类APP界面，其背后都离不开文本、图像、列表、超链接等核心元素的合理组合。

本章节将聚焦网页的基本元素，从理论到实践，带领大家逐步掌握这些元素的特性与应用方法，包括HTML的基本概念、网页文字与图像的控制方式等，帮助大家构建规范、美观且功能完整的网页。

任务列表

  • 任务2.1：编写第一个HTML页面

    基础

    在多元化信息的时代，网页是获取信息的重要渠道。网页是构成网站的基础，主要是由图像和文字等元素组成。本节主要按照HTML文档的基本格式，运用页面格式化标签来制作完成一个简单的网页，如图2-1所示。

    学习要点：

    • HTML文档基本格式与结构规范
    • 头部标签及其常用子标签
    • 标签分类（单标签与双标签）
    • 标签关系（嵌套关系与并列关系）
    • 特殊字符标签
    示例代码：HTML5文档基本结构
    <!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>我的第一个网页</title>
</head>
<body>
    <h1>欢迎学习网页设计与制作</h1>
    <p>这是我的第一个HTML页面</p>
</body>
</html>

    试一试：

    • 按照上述步骤，独立写出完整的 HTML 页面结构代码
    • 为网页设置自定义标题（如 "我的 HTML 学习笔记"）
    • 在代码中添加 3 条不同的注释（如说明标签功能、作者信息、创建日期）
    • 打开浏览器查看效果，验证注释是否会在页面中显示

  • 任务2.2：使用页面格式化标签组织课程概况

    核心

    网络的发展让信息传递更加便捷、快速，在网页中查找我们学习的课程资源成为我们获取信息的重要渠道。本案例将使用标题标签<hn>及属性、段落标签<p>、文本格式化标签<strong>等标签制作"课程概况"，如图2-3所示。

    学习要点：

    • 标题标签h1-h6及其对齐属性
    • 段落标签p及其对齐属性
    • 水平线标签hr及其属性
    • 换行标签br
    • 预格式化标签pre
    • 文本格式化标签strong、em、u、s等
    示例代码：课程概况页面
    <!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>课程概况</title>
</head>
<body>
    <!-- 页面主标题，居中对齐 -->
    <h1 align="center">网页设计与制作课程</h1>
    
    <!-- 水平线分隔，宽度80%，居中，红色细线条 -->
    <hr width="80%" align="center" color="#FF0000" size="1"/>
    
    <!-- 二级标题，课程简介 -->
    <h2>课程简介</h2>
    
    <!-- 段落文本，强调核心技术 -->
    <p>本课程旨在帮助学生掌握<strong>HTML5</strong>和<strong>CSS3</strong>的核心技术，能够独立设计并开发静态网页。</p>
    
    <!-- 二级标题，课程目标 -->
    <h2>课程目标</h2>
    
    <p>完成本课程后，学生将能够：</p>
    
    <!-- 无序列表展示课程目标 -->
    <ul>
        <li>理解Web标准和网页开发流程</li>
        <li>使用HTML5创建结构良好的网页</li>
        <li>使用CSS3美化网页样式</li>
    </ul>
</body>
</html>

    试一试：

    • 在页面中添加h3-h6各级标题，观察字体大小变化
    • 使用hr标签添加3条不同样式的水平线（不同颜色、粗细、宽度）
    • 在段落中使用br标签实现强制换行，对比与p标签的区别
    • 运用em、u、del等文本格式化标签设置文本样式
    • 使用pre标签展示一段带有原始排版的代码片段，观察显示效果

  • 任务2.3：使用图像展示课程特色

    实践

    一个精美、引人入胜的网页，往往包含很多图片。合理地使用图文混排，能够使枯燥的网页变得丰富多彩。本节将使用图像标签<img/>，并通过设置其"相对路径"，来制作图文混排的页面，其效果如图2-4所示。

    学习要点：

    • 常见图像格式（GIF、PNG、JPG、WebP）
    • img标签及其属性（src、alt、title、width、height等）
    • 图片路径：绝对路径和相对路径
    • 图像对齐方式（align属性）
    示例代码：图文混排页面
    <!doctype html>
<html>
<head>
    <meta charset="utf-8">
    <title>图文混排新闻</title>
</head>
<body>
    <!-- 插入图像：相对路径，宽度200px，居左，水平间距30px -->
    <img src="img/zhineng.png" width="200px" alt="人工智能相关图像" align="left" hspace="30" title="人工智能技术"/>
    
    <!-- 标题：微软雅黑字体，6号大小，深灰色 -->
    <h2><font face="微软雅黑" size="6" color="#545454">什么是人工智能？</font></h2>
    
    <!-- 分割线：灰色，1px粗 -->
    <hr color="#CCCCCC" size="1" />
    
    <!-- 发布时间：红色楷体 -->
    <p>
        <font color="#FF0000" face="楷体">
            <time datetime="2019-5-18">2019年05月18日</time>
        </font>
        
        <!-- 报道来源：蓝色楷体，前面加4个空格 -->
        <font color="#3366CC" face="楷体">
                综合科技报道
        </font>
    </p>
    
    <hr color="#CCCCCC" size="1" />
    
    <!-- 正文内容：2号大小，深灰色，强调核心功能 -->
    <p>
        <font size="2" color="#515151">
                人工智能是研究、开发用于模拟、延伸和扩展人的智能的理论、方法、技术及应用系统的一门新的技术科学，它能够通过学习
            <strong>自主决策</strong>、
            <strong>模式识别</strong>、
            <strong>分析数据</strong>，
            <font color="#FF0000">实现语言理解等复杂功能，</font>
            正深刻改变着人们的生活和社会的发展。
        </font>
    </p>
</body>
</html>

    试一试：

    • 从网上下载3张不同格式（GIF、PNG、JPG）的图像，放入"img"文件夹
    • 在页面中插入这3张图像，分别设置不同的width、height、border属性
    • 为每张图像添加alt替代文本和title提示文本，测试图像无法显示时的效果
    • 设置其中一张图像为右对齐（align="right"），观察文字环绕效果
    • 分别使用绝对路径和相对路径引用同一图像，对比两种路径的使用差异

  • 任务2.4：使用列表组织课程大纲

    核心

    我们在制作网站时，通常需要使用导航栏对网站信息进行分类，这样既可以方便我们对网站内容进行检索，也可以使网站的结构变得更清晰美观。本节将使用无序列表、有序列表制作"课程大纲"，其效果如图2-5所示。

    学习要点：

    • 有序列表ol、li及其type、start属性
    • 无序列表ul、li及其type属性
    • 定义列表dl、dt、dd
    • 列表嵌套
    示例代码：课程大纲页面
    <!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>课程大纲</title>
</head>
<body>
    <h1 align="center">课程大纲</h1>
    <hr width="80%" color="#3366CC" size="2"/>
    
    <!-- 有序列表：课程大纲，默认阿拉伯数字编号 -->
    <h3>课程模块</h3>
    <ol>
        <li>网页设计基础</li>
        <li>HTML5网页基本元素</li>
        <li>CSS3样式设计</li>
        <li>响应式网页布局</li>
    </ol>
    
    <!-- 无序列表：课程资源，方形项目符号 -->
    <h3>课程资源</h3>
    <ul type="square">
        <li>教材与讲义</li>
        <li>视频教程</li>
        <li>源代码</li>
    </ul>
    
    <!-- 定义列表：技术术语解释 -->
    <h3>技术术语</h3>
    <dl>
        <dt><strong>HTML5</strong></dt>
        <dd>最新的HTML标准，提供了更多语义化标签和API，增强了网页的功能与结构完整性。</dd>
        
        <dt><strong>CSS3</strong></dt>
        <dd>最新的CSS标准，提供了更多样式效果（如圆角、阴影、渐变）和灵活的布局方式（如Flex、Grid）。</dd>
    </dl>
</body>
</html>

    试一试：

    • 为有序列表设置type="I"，将编号改为大写罗马数字
    • 为无序列表设置type="circle"，将项目符号改为空心圆形
    • 创建嵌套列表：在有序列表的某个li中嵌套一个无序列表，展示子模块内容
    • 使用定义列表展示3个常用HTML标签（如img、a、p）的功能解释
    • 为有序列表设置start="5"，观察编号起始值的变化

  • 任务2.5：使用超链接完成页面跳转

    核心

    在浏览网站时，通过导航栏可以让访问者迅速找到所需要的资源区域，就如同出门旅行时旅客通过地图引导能够轻松到达目的地。导航栏相当于一个网站的地图、访问者可以通过导航纵观整个网站。本项目将运用列表和超链接的相关知识制作"导航栏的页面跳转"，效果如图所示。当鼠标指针点击相应的导航标题时，页面将从导航标题跳转至相应的页面。其效果如图2-6所示。

    学习要点：

    • 超链接标签a及其href、target属性
    • 锚点链接（页面内跳转）
    • 相对路径和绝对路径
    • 邮件链接和电话链接
    示例代码：课程导航页面
    <!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>课程首页</title>
    <style>
        /* 简单样式：让导航选项横向排列，添加间距 */
        ul { list-style: none; padding: 0; }
        li { display: inline-block; margin: 0 15px; }
        a { text-decoration: none; color: #3366CC; font-size: 18px; }
        a:hover { color: #FF0000; } /* 鼠标悬停时变色 */
    </style>
</head>
<body>
    <h1 align="center">网页设计与制作课程导航</h1>
    <hr width="90%" color="#3366CC" size="2"/>
    
    <!-- 无序列表构建导航栏 -->
    <ul align="center">
        <li><a href="index.html" target="_self">课程首页</a></li>
        <li><a href="kecheng_gaikuang.html" target="_blank">课程概况</a></li>
        <li><a href="rumen_gonglue.html" target="_blank">入门攻略</a></li>
        <li><a href="kecheng_xuexi.html" target="_blank">课程学习</a></li>
        <li><a href="jiaoxue_ziyuan.html" target="_blank">教学资源</a></li>
        <li><a href="zonghe_ceshi.html" target="_blank">综合测试</a></li>
    </ul>
</body>
</html>

    试一试：

    • 在"kecheng_xuexi.html"中添加页面内锚点，实现"课程大纲""学习重点""常见问题"之间的快速跳转
    • 添加外部链接：在导航栏中增加"MDN文档"选项，链接到https://developer.mozilla.org
    • 为超链接添加图像：将某个导航选项改为图像链接（如用"首页.png"替代"课程首页"文本）
    • 设置空链接：添加"更多功能"选项，使用href="#"作为占位链接
    • 修改target属性，测试_self和_blank的打开方式差异

  • 任务2.6：使用语义化标签创建课程页面框架

    进阶

    在网页设计中，如果按照从上到下的默认方式进行排版，网页会显得单调、混乱。这时就可以对网页进行布局，将部分模块有序排列，使网页的版面变得丰富、美观。本节将语义化标签header、导航栏nav、页面底部footer等来构建页面课程的整体框架。其效果如图2-7所示。

    学习要点：

    • 容器标签div和span
    • HTML5新增语义化标签（header、nav、main、section等）
    • 语义化标签的优势
    • 网页可访问性
    示例代码：语义化页面结构
    <!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>课程网站 - 语义化结构</title>
    <style>
        /* 简单样式优化：添加间距、边框，使区域更清晰 */
        body { width: 90%; margin: 0 auto; font-family: "微软雅黑"; }
        header, nav, main, aside, footer { margin: 10px 0; padding: 10px; border: 1px solid #eee; }
        nav ul { list-style: none; padding: 0; }
        nav li { display: inline-block; margin: 0 15px; }
        nav a { text-decoration: none; color: #333; }
        main { display: flex; }
        article { flex: 3; margin-right: 20px; }
        aside { flex: 1; }
        section { margin: 15px 0; padding: 10px; background: #f9f9f9; }
        footer { text-align: center; color: #666; }
    </style>
</head>
<body>
    <!-- 页眉：网站名称与标语 -->
    <header>
        <h1>网页设计与制作课程</h1>
        <p>学习HTML5和CSS3，掌握网页开发技能</p>
    </header>
    
    <!-- 导航栏：页面导航链接 -->
    <nav>
        <ul>
            <li><a href="#home">首页</a></li>
            <li><a href="#courses">课程</a></li>
            <li><a href="#resources">资源</a></li>
            <li><a href="#contact">联系</a></li>
        </ul>
    </nav>
    
    <!-- 核心内容区域 -->
    <main>
        <!-- 文章主体：课程教程内容 -->
        <article>
            <header>
                <h2>HTML5基础教程</h2>
                <p>发布于 2023年10月15日</p>
            </header>
            
            <!-- 文章小节：HTML5新特性 -->
            <section>
                <h3>HTML5新特性</h3>
                <p>HTML5引入了许多新的语义化标签（如<header>、<nav>、<main>等），替代了传统的<div>+id方式，让网页结构更清晰。同时新增了多媒体标签（<video>、<audio>）、表单新控件等功能，增强了网页的交互性与实用性。</p>
            </section>
            
            <!-- 文章小节：语义化标签的优势 -->
            <section>
                <h3>语义化标签的优势</h3>
                <p>使用语义化标签可以提高网页的可访问性，方便屏幕阅读器解析；同时利于搜索引擎优化（SEO），让搜索引擎更准确地识别网页核心内容；此外，语义化标签让代码更具可读性，便于团队协作与后期维护。</p>
            </section>
            
            <!-- 文章页脚：标签信息 -->
            <footer>
                <p>标签：HTML5, 语义化, 网页设计</p>
            </footer>
        </article>
        
        <!-- 侧边栏：相关课程推荐 -->
        <aside>
            <h3>相关课程</h3>
            <ul>
                <li><a href="#">CSS3样式设计</a></li>
                <li><a href="#">JavaScript基础</a></li>
                <li><a href="#">响应式网页设计</a></li>
            </ul>
        </aside>
    </main>
    
    <!-- 页脚：版权信息 -->
    <footer>
        <p>© 2023 网页设计与制作课程 版权所有</p>
    </footer>
</body>
</html>

    试一试：

    • 使用div标签替代语义化标签，通过id属性（如id="header"、id="nav"）实现相同的页面布局，对比两种方式的代码可读性
    • 在article中添加更多section小节，丰富文章内容
    • 在aside中添加图像链接（如课程封面图），优化侧边栏展示效果
    • 在footer中添加联系方式、备案号等信息
    • 尝试在main中嵌套article和section，进一步熟悉语义化标签的嵌套规则

总结与回顾

本章主要学习HTML5文件的基本结构，其中HTML5的结构中包含根标签<html>、首部标签<head>和主体标签<body>组成，其中的首部标签<head>包含的内容为网页的信息，而主题标签<body>所包含的内容会直接显示在网页上。其中HTML5保留了HTML4.01中一些常用标签，包括基础标签、文本格式标签、列表标签、图像标签、超链接标签、表格标签、容器标签等。同时HTML5也新增了具有明确语义的结构标签和格式标签。从而让网页的结构更加清晰。
