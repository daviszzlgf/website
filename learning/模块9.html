模块概述

学习目标

  • 知识目标：掌握CSS3高级选择器的类型及使用方法；了解CSS3新属性的用法；掌握线性渐变与径向渐变的属性与应用。
  • 能力目标：能综合运用高级选择器精准选择页面元素；能利用渐变等CSS3新属性制作质感UI组件，创建丰富的背景效果。
  • 素养目标：践行工匠精神，提升职业素养水平，注重代码的可维护性与设计细节。

建议学时

4学时

核心模块

在网页设计中，细节决定成败。要打造真正具有吸引力与专业感的用户界面，离不开对CSS高级特性的娴熟运用。本项目将展示如何通过高级选择器实现精准的样式控制，并借助强大的渐变工具营造丰富的视觉层次，以简洁高效的代码优化整体用户体验。

任务列表

  • 任务9.1：认识CSS的高级选择器

    基础

    任务描述：

    在掌握了基础选择器后，我们需要应对更复杂的结构和交互。本任务将系统学习属性选择器、伪类、伪元素及组合选择器，通过分析典型代码，理解它们的作用与场景，为精准控制样式奠定基础。

    知识链接：

    9.1.1 属性选择器
    • [attr]：选择包含指定属性的元素
    • [attr=value]：属性值完全匹配
    • [attr~=value]：属性值包含词汇（空格分隔）
    • [attr^=value]：属性值以指定值开头
    • [attr$=value]：属性值以指定值结尾
    • [attr*=value]：属性值包含指定子串
    9.1.2 状态伪类 & 结构伪类
    • :hover, :focus, :active, :checked 等交互状态
    • :first-child, :last-child, :nth-child(n) 结构位置
    • :not(selector) 否定伪类
    9.1.3 伪元素
    • ::before / ::after 插入生成内容
    • ::first-letter, ::first-line, ::selection 文本装饰
    9.1.4 组合选择器
    • 后代 (空格) 、子元素 (>) 、相邻兄弟 (+) 、通用兄弟 (~)

    常用选择器分类

    属性选择器
    • [disabled]
    • [type="submit"]
    • [class^="btn-"]
    状态伪类
    • :hover
    • :focus
    • :checked
    结构伪类
    • :first-child
    • :nth-child(odd)
    • :not(:last-child)
    伪元素
    • ::before
    • ::after
    • ::first-line
    高级选择器示例
    /* 属性选择器 */
a[target="_blank"] { color: red; }
[class^="icon-"] { font-size: 1.2rem; }

/* 伪类 */
li:first-child { font-weight: bold; }
tr:nth-child(even) { background: #f2f2f2; }
input:focus { border-color: #3498db; }

/* 伪元素 */
blockquote::before { content: "“"; font-size: 2rem; }
p::first-letter { font-size: 2rem; float: left; }

/* 组合 */
nav > ul > li { display: inline-block; }
h2 + p { margin-top: 0; }

  • 任务9.2：理解与应用CSS选择器优先级

    核心

    任务描述：

    样式冲突是日常开发中的常见问题。本任务将深入剖析选择器优先级（特异性）的计算规则，学会诊断样式覆盖问题，编写更健壮的CSS。

    优先级计算规则 (a,b,c,d)：

    • 千位(a)：内联样式，有则为1
    • 百位(b)：ID选择器数量
    • 十位(c)：类、属性、伪类数量
    • 个位(d)：元素、伪元素数量

    从左到右逐级比较，数值大的优先级高。相同则后定义的生效。

    !important 与继承

    !important 声明具有最高优先级，但应谨慎使用。继承的样式优先级最低。

    优先级计算示例
    /* 1. #header .nav-item  → (0,1,1,0) */
/* 2. ul.menu > li:first-child → (0,0,2,2) */
/* 3. div p → (0,0,0,2) */
/* 4. .btn.primary:hover → (0,0,3,0) */

// 内联 style="color:red" 优先级最高 (1,0,0,0)

  • 任务9.3：使用高级选择器和CSS3新属性精准控制样式

    应用

    任务描述：

    以课程网站为例，实现当前页面高亮、导航悬停反馈、面包屑分隔符、标题装饰下划线。这些效果完全依靠高级选择器完成，无需增加额外类名。

    实现要点：

    • 导航高亮：nav a.active 与 :hover
    • 面包屑分隔符：.breadcrumb li:not(:last-child)::after { content: "›"; }
    • 标题下划线：.page-header h1::after 生成装饰线
    • 组合选择器隔离样式：nav > ul > li

    当前页面头部、面包屑已经应用了这些样式，可查看源码学习。

  • 任务9.4：使用渐变属性打造质感背景

    设计

    任务描述：

    渐变可以替代单一背景色，使页面更生动。本任务学习线性渐变、径向渐变、重复渐变，并应用于导航、卡片、按钮等。

    常用渐变函数：

    • linear-gradient(direction, color1, color2...)
    • radial-gradient(shape at position, color1, color2...)
    • repeating-linear-gradient / repeating-radial-gradient

    应用示例：

    渐变核心代码
    /* 线性渐变 */
background: linear-gradient(135deg, #2c3e50, #3498db);

/* 径向渐变 */
background: radial-gradient(circle at 30% 30%, #fff, #3498db);

/* 重复渐变 (条纹) */
background: repeating-linear-gradient(45deg, 
  #3498db 0px, #3498db 10px, 
  #2980b9 10px, #2980b9 20px);

总结与回顾

通过本项目我们系统掌握了CSS高级选择器与渐变属性，实现了样式的精准控制与质感提升。学会了运用选择器精准定位元素，简化了HTML结构。同时，通过灵活使用线性与径向渐变，我们创造出丰富的背景视觉效果，显著提升了界面的专业性与用户体验，锤炼了精细化设计与代码维护的工匠精神，为开发现代化网页奠定了坚实基础。

一、选择题

  1. 以下哪个选择器可以选择所有带有target属性的链接？ A. a[target] B. a.target C. a>target D. a:target
  2. 以下哪个伪类用于选择被访问后的链接？ A. :hover B. :active C. :visited D. :link
  3. 创建一个从左到右的线性渐变，应该使用？ A. to top B. to bottom C. to left D. to right
  4. 以下哪个伪元素用于在元素内容之后插入内容？ A. ::before B. ::after C. ::first-line D. ::first-letter
  5. 在CSS中，哪个函数用于创建径向渐变？ A. linear-gradient B. radial-gradient C. conic-gradient D. gradient-radial
  6. 以下哪个选择器会选择所有直接子元素？ A. div p B. div > p C. div + p D. div ~ p
  7. 哪个选择器优先级最高？ A. .container .btn B. #header .btn C. btn D. div#header.btn
  8. CSS变量的定义语法是？ A. $变量名:值 B. --变量名:值 C. var(变量名):值 D. @变量名:值

  3. 在径向渐变中，_______参数用于设置渐变的形状（circle或ellipse）。
  4. 用于选择作为其父元素的第一个子元素的选择器是_______。
  5. CSS变量的作用域分为全局和_______。

三、判断题

  1. 属性选择器[class^="btn"]会选择所有class属性以"btn"开头的元素。（ ）
  2. 线性渐变只能从上到下或从左到右。（ ）
  3. :nth-child(2n)选择器会选择所有偶数位置的子元素。（ ）
  4. 径向渐变的默认形状是圆形。（ ）
  5. 伪元素::first-letter只能应用于块级元素。（ ）

四、优先级计算

  • 1. div.container { color: blue; } → (0,0,1,1)
