模块4：CSS3盒子模型

  • 首页

  • 模块4：CSS3盒子模型

模块概述

学习目标

  • 知识目标：掌握盒子模型四层结构、标准与怪异盒模型计算规则、padding/border/margin 语法、外边距合并、元素类型转换、背景延伸与溢出处理、Flex/Grid 基础原理。
  • 能力目标：能运用盒子模型精确控制尺寸与间距；能解决外边距合并、inline-block 间隙；能使用display转换元素类型；能结合box-sizing统一布局规则；能运用Flex/Grid完成多内容卡片布局。
  • 素养目标：建立模块化、标准化的CSS编写习惯；在布局调试中主动分析盒模型计算问题；理解盒子模型作为CSS布局核心，建立系统化知识体系；提升团队协作中代码的一致性与可复用性。

建议学时

12学时

基础核心模块

打开任何一个网页，按下F12键，你会发现那些色彩斑斓的导航栏、卡片、按钮和图片，本质上都是由"方方正正的盒子"构成的。这种"四层包裹"规则——内容(content)、内边距(padding)、边框(border)和外边距(margin)——构成了Web前端开发的第一块基石。本模块将从"一个盒子"出发，逐层解析、逐像素测量，带你完成从基础认知到精确计算的盒子模型进阶之旅。

核心任务

  • 任务4.1：页面结构划分与盒子模型

    基础

    从"盒子"视角将空白页面划分为container、header、content、sidebar、footer五个区域，使用语义化类名标识。通过背景色、边框直观呈现层级关系，掌握盒子模型四层结构的基础应用。

    学习要点：

    • 盒子模型四层结构：content / padding / border / margin
    • 背景延伸控制：background-clip / background-origin
    • 溢出处理：overflow属性（visible/hidden/scroll/auto）
    • 尺寸计算规则：标准盒模型(content-box) vs 怪异盒模型(border-box)
    • 页面样式重置：* { box-sizing: border-box; margin:0; padding:0; }
    示例代码：全局重置与盒子模型
    /* 统一盒子模型，清除默认边距 */
* {
    box-sizing: border-box;
    margin: 0;
    padding: 0;
}

/* 标准盒模型：width 仅指内容区 */
.standard {
    box-sizing: content-box;
    width: 200px;
    padding: 20px;
    border: 5px solid #333;
    /* 实际宽度 = 200 + 40 + 10 = 250px */
}

/* 怪异盒模型：width 包含 padding 和 border */
.weird {
    box-sizing: border-box;
    width: 200px;
    padding: 20px;
    border: 5px solid #333;
    /* 实际宽度 = 200px (内容区被压缩为 200-40-10=150px) */
}

  • 任务4.2：横向导航栏布局实现

    布局

    运用盒子模型的padding、border、margin属性，结合元素类型转换，将垂直列表转换为水平导航栏。控制链接宽度、悬停效果及间距，实现规范美观的横向导航栏。

    学习要点：

    • 内边距padding（复合写法与方向控制）
    • 边框border（宽度、样式、颜色、圆角、阴影）
    • 外边距margin（复合写法、负值应用）
    • 外边距合并（Collapsing）及解决方案
    标准盒模型 content-box

    content-box

    width = 内容区宽度，padding和border额外增加总宽

    怪异盒模型 border-box

    border-box

    width = 内容区+padding+border，内边距和边框向内挤压

    外边距合并

    Margin Collapsing

    垂直相邻块级外边距取较大值，而非相加

    示例代码：横向导航栏
    /* 清除列表默认样式 */
ul {
    list-style: none;
    margin: 0;
    padding: 0;
}
li {
    display: inline-block;  /* 横向排列 */
}
a {
    display: inline-block;
    padding: 8px 15px;
    margin-right: 10px;
    border: 1px solid transparent;
    border-radius: 4px;
    text-decoration: none;
    color: #333;
}
a:hover {
    background: #f0f0f0;
    border-color: #3498db;
}

  • 任务4.3：元素类型转换

    类型转换

    通过display属性改变元素默认显示类型，实现卡片横向排列。掌握块级、行内、行内块的特点，解决inline-block间隙问题，完成静态卡片组布局。

    学习要点：

    • 块级元素（block）特征
    • 行内元素（inline）特征
    • 行内块元素（inline-block）特征
    • inline-block间隙产生原因与解决方案（父元素font-size:0）
    示例代码：inline-block 间隙消除
    <style>
    .parent {
        font-size: 0;  /* 消除空白节点间隙 */
    }
    .child {
        display: inline-block;
        width: 100px;
        height: 100px;
        font-size: 16px;  /* 恢复字体 */
        background: #3498db;
        margin-right: 10px;
    }
</style>
<div class="parent">
    <div class="child">卡片1</div>
    <div class="child">卡片2</div>
    <div class="child">卡片3</div>
</div>

  • 任务4.4：多内容卡片布局与尺寸精确控制

    现代布局

    学习Flex弹性布局与Grid网格布局，解决传统inline-block在复杂卡片布局中的局限。实现一维横向均布与二维行列精确控制，完成信息丰富的企业级卡片布局。

    学习要点：

    • Flex容器属性（display:flex, flex-direction, justify-content, align-items）
    • Flex项目属性（flex-grow, flex-shrink, flex-basis）
    • Grid容器属性（grid-template-columns/rows, gap, repeat, fr单位）
    • Grid项目跨列/跨行（grid-column / grid-row）
    示例代码：Grid三列卡片布局
    .card-container {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(280px, 1fr));
    gap: 20px;
}
.card {
    background: white;
    border-radius: 8px;
    box-shadow: 0 2px 8px rgba(0,0,0,0.1);
    padding: 20px;
    transition: transform 0.3s;
}
.card:hover {
    transform: translateY(-5px);
}

综合实训

综合实训：数码商城商品列表页

实战

开发数码商城商品列表页的静态页面，包含顶部导航、筛选栏、商品列表、底部版权。要求精准控制元素间距与尺寸，满足基础响应式适配。

任务要求：

  • 顶部导航栏：Flex布局实现两端对齐与垂直居中，内边距上下12px、左右20px
  • 筛选栏：inline-block布局实现横向排列（宽度不足自动换行），筛选项左右间距8px、底部间距10px，内边距上下5px、左右12px
  • 商品列表区：Grid布局，列宽最小280px、最大等分剩余空间，卡片间距20px
  • 商品卡片内部：Flex垂直布局，内边距15px，图片宽100%、高200px
  • 底部版权区：文字水平居中，内边距上下20px
商品列表Grid参考
.product-grid {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));
    gap: 20px;
    padding: 20px 0;
}
.product-card {
    display: flex;
    flex-direction: column;
    border-radius: 8px;
    overflow: hidden;
    box-shadow: 0 2px 5px rgba(0,0,0,0.1);
}
.product-card img {
    width: 100%;
    height: 200px;
    object-fit: cover;
}
.card-content {
    padding: 15px;
    flex: 1;
}
