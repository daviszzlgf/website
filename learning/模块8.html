模块概述

学习目标

  • 知识目标：系统理解静态网站开发全流程规划方法，完成从主题定位、结构设计到语义化框架搭建。
  • 能力目标：综合运用HTML5、CSS3及布局技术，独立开发、测试并发布一个功能完整的响应式网站。
  • 素养目标：将代码规范内化为开发习惯，培养团队协作意识，建立系统工程化与职业化项目思维。

项目时长

8学时

完整项目开发流程

本模块作为《网页设计与制作》课程的压轴环节，将前七个模块的知识点通过一个完整的“精品课程网站”项目进行有机整合。你将亲身体验从项目规划、设计、编码、测试到最终交付的完整工作流程，在实战中巩固代码规范，培养团队协作意识，并初步建立系统工程化项目思维。最终产出的不仅是一个可运行的网站，更是一份展现学习成果与职业潜力的“作品集”。

任务列表（7个子任务）

  • 任务8.1：项目启动与站点结构设计

    规划

    创建项目目录结构，明确主题与核心模块，运用HTML5语义化标签搭建首页框架。

    学习要点：

    • 确定网站主题（精品课程）及功能页面（首页、概况、资源、测试等）
    • 创建规范的目录结构：css/、images/、pages/、index.html
    • 使用<header>、<nav>、<main>、<footer>搭建语义化骨架

    项目目录结构示例

    标准静态项目结构
    website/
       css/
         style.css
       images/
       pages/
         overview.html
         resources.html
       index.html

    根文件夹包含所有资源，实现结构与样式分离。

    HTML5语义化骨架
    <header>...</header>
    <nav>...</nav>
    <main>...</main>
    <footer>...</footer>

    替代无序div堆砌，提升可读性与SEO。

    index.html 基础框架
    <!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>网页设计与制作精品课程</title>
    <link rel="stylesheet" href="css/style.css">
</head>
<body>
    <header>
        <h1>网页设计与制作精品课程</h1>
        <nav>
            <ul>
                <li><a href="index.html">首页</a></li>
                <li><a href="pages/overview.html">课程概况</a></li>
                <li><a href="pages/resources.html">教学资源</a></li>
            </ul>
        </nav>
    </header>
    <main>
        <section>
            <h2>课程简介</h2>
            <p>...</p>
        </section>
    </main>
    <footer>
        <p>© 2026 网页设计与制作课程组</p>
    </footer>
</body>
</html>

  • 任务8.2：首页布局与视觉规范实现

    核心

    基于HTML骨架，运用CSS变量、Flexbox及媒体查询实现首页布局与响应式原型。

    学习要点：

    • 在:root中定义色彩体系与字体变量
    • 使用Flexbox构建导航栏与特色卡片区
    • 设置.container通用容器与响应式断点

    CSS变量与Flex示例

    :root 变量定义

    :root {
      --primary: #2c3e50;
      --secondary: #3498db;
      --font-family: 'Segoe UI', sans-serif;
    }

    Flex导航栏

    .site-header .container {
      display: flex;
      justify-content: space-between;
      align-items: center;
    }
    style.css 核心样式
    /* 视觉规范变量 */
:root {
    --primary: #2c3e50;
    --secondary: #3498db;
    --light: #ecf0f1;
    --background: #f9f9f9;
    --font-family: 'Segoe UI', Tahoma, sans-serif;
}

/* 通用容器 */
.container {
    width: 90%;
    max-width: 1200px;
    margin: 0 auto;
    padding: 0 15px;
}

/* 特色卡片 Flex 布局 */
.feature-grid {
    display: flex;
    justify-content: space-between;
    flex-wrap: wrap;
}
.feature-card {
    flex: 1 1 300px;
    margin: 1rem;
    background: white;
    border-radius: 8px;
    box-shadow: 0 4px 8px rgba(0,0,0,0.1);
}

/* 响应式 */
@media (max-width: 768px) {
    .site-header .container {
        flex-direction: column;
    }
    .feature-grid {
        flex-direction: column;
    }
}

  • 任务8.3：课程资源中心页面开发

    应用

    创建resources.html，以卡片网格形式展示课程资源（PPT、视频、源码等），实现自适应排列。

    学习要点：

    • 使用flex-wrap + gap构建响应式卡片网格
    • 设计资源卡片内部结构（图标、标题、元数据、按钮）
    • 复用全局样式并扩展特有样式

  • 任务8.4：表单交互与综合测试页面

    交互

    构建课程评价表单(feedback.html)和综合测试入口(test.html)，实现表单验证与样式美化。

    学习要点：

    • 使用<form>、<fieldset>、<input>、<textarea>等元素
    • 添加required属性进行原生验证
    • 统一表单控件样式，与网站视觉规范一致
    评价表单结构
    <form>
      <fieldset>
        <legend>基本信息</legend>
        <label>课程名 *</label>
        <input type="text" required>
      </fieldset>
      <fieldset>...</fieldset>
      <button type="submit">提交评价</button>
    </form>
    表单核心CSS
    .form-row input[type="text"],
.form-row select,
.form-row textarea {
    width: 100%;
    padding: 0.8rem;
    border: 1px solid #ddd;
    border-radius: 4px;
}
.form-row input:focus {
    border-color: var(--secondary);
    box-shadow: 0 0 0 2px rgba(52,152,219,0.2);
}
.radio-group {
    display: flex;
    gap: 1.2rem;
    flex-wrap: wrap;
}

  • 任务8.5：响应式适配与跨浏览器测试

    优化

    运用浏览器开发者工具进行多设备模拟，完善媒体查询，确保在主流浏览器中表现一致。

    学习要点：

    • 使用meta viewport设置视口
    • 针对768px及480px设置断点优化
    • 测试Chrome、Firefox、Edge兼容性
    媒体查询示例
    /* 平板及以下 */
@media (max-width: 768px) {
    .feature-grid { flex-direction: column; }
    .main-nav ul { flex-wrap: wrap; }
}

/* 小手机 */
@media (max-width: 480px) {
    .main-nav ul { flex-direction: column; width: 100%; }
    .main-nav li { margin: 0.3rem 0; }
}

  • 任务8.6：网站部署与上线发布

    运维

    选择虚拟主机或本地Nginx服务器，注册域名，通过FTP上传，完成线上测试。

    学习要点：

    • 区分虚拟主机、VPS、云服务器适用场景
    • 域名注册与A记录解析
    • 使用FileZilla上传文件到服务器根目录

    💡 部署示意：本地website/ → 服务器wwwroot/，配置nginx root路径，访问http://你的域名/index.html

  • 任务8.7：项目文档撰写与成果展示

    总结

    编写README.md项目文档，制作展示PPT或海报，进行项目汇报。

    学习要点：

    • 文档结构：概述、设计说明、实现细节、测试报告、总结
    • 使用Markdown语法，配合截图展示成果
    • 准备3-5分钟演示，突出技术亮点
