模块概述

学习目标

  • 知识目标：掌握CSS语法规则和三种引用方式；了解常见的字体、文本、背景和边框属性的含义；理解CSS优先级和继承规则。
  • 能力目标：能够理解并应用CSS样式表，熟练使用CSS选择器；能够使用CSS控制字体样式属性、文本外观属性、背景属性和边框属性；能够为“精品课程站点”建立初步的视觉规范。
  • 素养目标：树立代码规范意识，理解样式规则在构建用户体验中的关键作用，在实现中坚守工匠精神，避免滥用样式。

建议学时

16学时

基础模块

完成网页结构搭建后，前端开发进入视觉设计阶段。本模块系统讲解CSS样式设计，通过代码实现专业的网页视觉效果。包括CSS语法基础，文本样式的灵活运用，初级选择器的掌握使用，以及背景、边框装饰的精细处理。通过“精品课程站点”的实践案例，详细演示如何将设计理念转化为具体的样式代码，实现从基础内容到精致界面的转变。

任务列表

  • 任务3.1：建立课程网站样式基础

    基础

    作为整个网站美化的起点，学习CSS语法规则，掌握样式表的引用方式，为后续课程网站的美化打下基础。通过对课程网站的简单元素添加基础样式，直观感受CSS如何改变网页的视觉呈现。

    学习要点：

    • CSS简介与语法规则：选择器{属性:值;}
    • 三种样式表引用方式：内联样式、内部样式表、外部样式表
    • 样式应用：通过行内样式设置字体颜色、内部样式设置背景、外部样式清除默认边距

    CSS语法规则示例

    选择器 属性 属性值 作用说明
    p color red 设置段落文字为红色
    h1 font-size 24px 设置一级标题字号为24像素
    body background-color #f9f9f9 设置页面背景色
    示例代码：三种样式引用方式
    <!-- 1. 内联样式 -->
<p style="color:red;font-size:16px;">红色段落</p>

<!-- 2. 内部样式表（放在head中） -->
<style>
  body { background-color: #f0f0f0; }
  h1 { color: navy; }
</style>

<!-- 3. 外部样式表（link标签） -->
<link rel="stylesheet" href="css/style.css">

    为deepseek.html页面添加行内样式、内部样式以及外部样式，初始化该页面的CSS样式。

  • 任务3.2：设计网站字体排版

    核心

    为“课程概况”页面建立完整的字体排版规范。学习CSS文本属性，掌握字体类型、字号层级、字重变化与行高设置的原理与方法，打造层次清晰、阅读舒适的专业化界面。

    学习要点：

    • 字体样式属性：font-family, font-size, font-weight, font-style
    • 文本样式属性：text-align, text-decoration, line-height, text-shadow
    • CSS颜色体系：十六进制、RGB、RGBA、HSL、HSLA

    常用字体/文本属性示例

    属性 可选值/示例 说明
    font-family "宋体", Arial, sans-serif 设置字体类型
    font-size 16px, 1.2rem, large 字体大小
    font-weight bold, 700, normal 字体粗细
    text-align center, left, right 水平对齐
    line-height 1.6, 24px 行高
    示例代码：字体排版设置
    body {
    font-family: 'Segoe UI', Tahoma, sans-serif;
    color: #333;
    line-height: 1.6;
}
h1 {
    font-size: 2.5rem;
    font-weight: bold;
    text-align: center;
    text-shadow: 2px 2px 4px rgba(0,0,0,0.2);
}
p {
    font-size: 1rem;
    text-decoration: none;
    color: #2c3e50;
}

    试一试：

  • 任务3.3：使用选择器精确控制样式

    进阶

    学习CSS选择器的核心知识，掌握如何精准定位并控制页面中的特定元素。通过标签选择器、类选择器、ID选择器等不同类型选择器的实践应用，实现对“课程评价”页面样式的精确控制，避免样式冲突。

    学习要点：

    • 基础选择器：元素选择器、类选择器、ID选择器、通配符选择器
    • 选择器命名规范与优先级（权重计算）
    • CSS三大特性：继承性、层叠性、优先级

    选择器优先级示例

    选择器类型 示例 权重（优先级）
    通配符选择器 * 0,0,0,0
    元素选择器 p, h1, div 0,0,0,1
    类选择器 .container, .topnav 0,0,1,0
    ID选择器 #header, #container 0,1,0,0
    行内样式 style="..." 1,0,0,0
    示例代码：选择器应用
    /* 元素选择器 */
header { background-color: #2c3e50; }

/* 类选择器 */
.topnav { color: white; text-decoration: none; }

/* ID选择器 */
#container { width: 90%; margin: 0 auto; }

/* 通配符选择器 */
* { margin: 0; padding: 0; }

    试一试：

    在deepseek.html中使用内部样式，选用通配符、ID、类、标签选择器修饰页面样式。

  • 任务3.4：页面元素的样式深入设计

    进阶

    学习CSS中背景样式、边框样式与列表样式的综合应用。通过掌握背景图像控制、边框效果设计及列表符号优化等技能，为“课程概况”页面的各类元素添加恰当的视觉装饰，提升页面的美观度与专业感。

    学习要点：

    • 背景样式：background-color, background-image, background-size, background-repeat
    • 边框样式：border-width, border-style, border-color, border-radius
    • 列表样式：list-style-type, list-style-image
    • 圆角与阴影：box-shadow

总结与回顾

本项目讲解了CSS基础语法以及三种引用方式，通过色彩心理学原理构建了专业的网站配色方案，接着建立了科学的字体排版规范，然后掌握了CSS选择器的精准定位技巧，最后深入运用了背景样式、边框样式以及颜色值的使用进行视觉装饰。每个任务都以“精品课程站点”为实践载体，将理论知识转化为可操作的样式代码。本项目的学习不仅让我们掌握了CSS的核心样式属性，更重要的是培养了将设计意图转化为代码实现的系统化思维，为后续复杂页面的样式设计奠定了坚实基础。

  • CSS语法规则与三种引用方式

习题

一、选择题

  • 1. CSS是（C）的缩写。 A. Colorful Style Sheets B. Computer Style Sheets C. Cascading Style Sheets D. Creative Style Sheets
  • 2. 以下不是CSS引用方式的是（D）。 A. 行内样式 B. 内部样式 C. 外部样式 D. 尾部样式
  • 3. 引用外部样式表的格式是（B）。 A. <style src="mystyle.css"> B. <link rel="stylesheet" type="text/css" href="mystyle.css"> C. <stylesheet>mystyle.css</stylesheet> D. <style type="text/css" href="mystyle.css">
  • 4. 引用内部样式表的元素应该放在（C）。 A. HTML文档的开始 B. HTML文档的结束 C. <head>元素中 D. <body>元素中
  • 5. 下列关于颜色的表述，不能显示为红色的是（D）。 A. red B. #ff0000 C. rgb(255,0,0) D. hsl(120,100%,50%)

二、名词解释

  • CSS：层叠样式表，用于控制网页元素外观的标记语言。
  • 内部样式表：写在HTML文档<head>的<style>标签中的CSS代码。
  • 类选择器：以点号（.）开头，通过class属性匹配元素的选择器。

三、判断题

  • 1. 在CSS中，ID选择器的优先级高于类选择器。 （√）
  • 2. CSS选择器不区分大小写，属性和值也不区分大小写。 （√）
  • 3. 在外部样式表引用.css文件时，<link>元素当中rel用来定义所链接的外部样式表文件的URL。 （×，rel定义关系）
  • 4. background-repeat属性的默认值是no-repeat。 （×，默认repeat）
