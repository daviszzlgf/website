  • 首页

  • 模块6：表格与表单

模块概述

学习目标

  • 知识目标：掌握CSS3中超链接4种核心状态的样式设置规则；掌握CSS3列表样式核心属性的使用方法；掌握HTML表格的创建与美化；掌握表单核心控件及布局。
  • 能力目标：能够运用CSS3设置超链接不同状态；使用列表样式规整信息；独立创建数据表格并美化；制作完整表单并优化样式。
  • 素养目标：树立数据安全隐私意识，遵守用户信息保护准则，遵循个人信息保护法及安全规范。

建议学时

12学时

理论+实践

网页设计中，表格是结构化数据呈现的核心载体，可清晰展示各类规整数据；表单是实现用户与网页交互的关键工具，用于收集各类用户信息。本模块围绕超链接多状态样式、列表自定义样式、表格创建与美化、表单设计与常用控件应用，助你系统掌握相关核心技术，树立数据安全与隐私保护意识，独立完成超链接、列表、表格及表单的相关开发。

任务列表

  • 任务6.1：设置导航栏超链接不同状态

    基础

    为学习资讯页面制作导航栏，为导航栏中的超链接设置不同交互状态的样式，包括未访问（默认）、已访问、悬停、点击瞬间四种状态，使导航栏在用户进行不同操作时呈现清晰的视觉反馈，并高亮显示当前页。

    知识要点：

    • 超链接伪类 :link :visited :hover :active
    • LVHA 书写顺序
    • 当前页高亮样式 (.active)
    • 行内元素转块级扩大点击区域

    导航栏状态演示

    示例代码：超链接伪类
    /* LVHA 顺序 */
nav a:link { color: #FFFFFF; text-decoration: none; }
nav a:visited { color: #d1e7ff; }
nav a:hover { background-color: #ffffff; color: #2f658d; border-radius: 4px; }
nav a:active { background-color: #194060; color: #f8fcff; }
nav a.active { background-color: #4a90e2; color: #ffffff; border-radius: 4px; }

    任务实施：

    • 步骤1：搭建包含nav和ul的HTML结构
    • 步骤2：设置导航基本样式（浮动横向）
    • 步骤3：定义:link、:visited、:hover、:active
    • 步骤4：为当前页添加active类高亮

  • 任务6.2：设置"学习资讯"页面列表样式

    基础

    制作学习网站中"学习资讯"页面，使用列表标签与CSS3列表样式属性规整展示"最新资讯分类""课程通知""学习干货"等模块，让信息结构清晰、视觉层次分明。

    知识要点：

    • list-style-type (disc, decimal, square等)
    • list-style-position (inside/outside)
    • list-style-image 自定义符号
    • 复合属性 list-style

    列表样式演示

    任务实施：

    • 步骤1：构建三个列表模块（ul/ol）
    • 步骤2：为不同模块设置差异化项目符号
    • 步骤3：调整间距、字体、悬浮效果
    • 步骤4：添加"查看更多"、"新"标签等（试一试）

  • 任务6.3：使用表格展示学员数据

    核心

    基于学员数据，学习HTML创建规范表格，通过CSS3样式优化视觉呈现，最终完成"学员数据表"，掌握表格在结构化数据展示中的实际应用及CSS3美化技巧。

    知识要点：

    • 表格标签：table, caption, thead, tbody, tr, th, td
    • 合并单元格：colspan, rowspan
    • CSS表格属性：border-collapse, padding, text-align
    • 隔行变色、悬停高亮

    学员表格演示

    border-collapse: collapse;

    padding: 8px;

tbody tr:hover { background-color: #b0daff; }

    任务实施：

    • 步骤1：创建table>caption+thead+tbody结构
    • 步骤2：使用colspan/rowspan合并单元格
    • 步骤3：通过CSS设置边框、内边距、对齐
    • 步骤4：添加表头背景、悬停效果

  • 任务6.4：设计用户注册表单

    核心

    使用HTML表单功能，设计一套用户注册表单。包含核心功能字段（用户名、邮箱、密码），实现"注册表单"功能，并通过CSS美化表单样式。

    知识要点：

    • 表单域 form (action, method)
    • input类型：text, password, email, radio, checkbox
    • label, placeholder, required
    • 按钮 button vs input submit

    注册表单演示



    男 女
    示例代码：注册表单结构
    <form method="post">
  <label for="username">用户名：</label>
  <input type="text" id="username" required placeholder="用户名">
  
  <label for="email">邮箱：</label>
  <input type="email" id="email" required>
  
  <label for="password">密码：</label>
  <input type="password" id="password" minlength="6" required>
  
  <button type="submit">注册</button>
</form>

    任务实施：

    • 步骤1：创建form容器，设置method="post"
    • 步骤2：添加文本、密码、邮箱输入框及单选按钮
    • 步骤3：使用label绑定控件，增加placeholder
    • 步骤4：CSS美化（圆角、内边距、悬浮效果）

  • 任务6.5：实现注册表单个性化选项

    进阶

    在注册页面基础上，添加个性化选项：兴趣爱好（复选框）、所在地区（下拉列表）、个人简介（文本域），并对新增元素进行美化，同时遵循数据安全规范，仅收集必要信息。

    知识要点：

    • 复选框 (checkbox) 多选
    • 下拉列表 select + option + optgroup
    • 文本域 textarea (rows/cols)
    • 个人信息保护法：最小必要原则

    个性化选项演示

    阅读 运动 音乐

    示例代码：个性化控件
    <!-- 复选框 -->
<input type="checkbox" name="hobby" value="reading"> 阅读

<!-- 下拉分组 -->
<select name="city">
  <optgroup label="热门城市">
    <option value="zhengzhou">郑州</option>
  </optgroup>
</select>

<!-- 文本域 -->
<textarea name="intro" rows="3" cols="40" placeholder="个人简介"></textarea>

    任务实施：

    • 步骤1：添加复选框组，name相同为数组
    • 步骤2：添加select下拉列表，使用optgroup分组
    • 步骤3：添加textarea用于多行文本
    • 步骤4：统一CSS美化，保持风格一致

归纳与总结

通过本模块学习，我们掌握了：超链接四种状态的伪类设置（LVHA顺序）；列表样式的核心属性（list-style-type, position, image）；表格的创建（table, thead, tbody, colspan/rowspan）及CSS美化（边框合并、内边距、悬停）；表单的结构（form, label, input, select, textarea）以及常用控件类型。同时，我们了解了数据安全规范（个人信息保护法、最小必要原则），为后续开发功能完善的交互网页打下基础。
