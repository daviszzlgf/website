HTML基础规范

文档类型声明

始终使用HTML5的文档类型声明：

<!DOCTYPE html>

语言属性

明确指定文档的语言：

<html lang="zh-CN">

字符编码

在<head>中首先声明字符编码：

<meta charset="UTF-8">

Viewport设置

为移动设备正确设置viewport：

<meta name="viewport" content="width=device-width, initial-scale=1.0">

HTML文档结构

良好示例：
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>页面标题</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <!-- 页面内容 -->
</body>
</html>

HTML属性规范

属性使用双引号，不要使用单引号：

不良示例：
<div class='container'>
良好示例：
<div class="container">

布尔属性可以省略值：

<input type="checkbox" checked>
<input type="text" disabled>

HTML语义化

使用语义化标签提高可访问性和SEO：

<header>
  <nav><!-- 导航 --></nav>
</header>
<main>
  <article>
    <section>
      <h1>文章标题</h1>
      <p>文章内容</p>
    </section>
  </article>
  <aside><!-- 侧边栏 --></aside>
</main>
<footer>
  <p>版权信息</p>
</footer>
提示： 避免滥用<div>标签，优先使用语义化标签如<header>、<nav>、<main>、<article>、<section>、<aside>和<footer>。

CSS基础规范

CSS书写顺序

建议按照以下顺序组织CSS属性：

.selector {
  /* 布局属性 */
  display: block;
  position: absolute;
  top: 0;
  left: 0;

  /* 盒模型属性 */
  width: 100px;
  height: 100px;
  margin: 10px;
  padding: 10px;
  border: 1px solid #ccc;

  /* 文本属性 */
  font-family: Arial, sans-serif;
  font-size: 16px;
  line-height: 1.5;
  color: #333;

  /* 视觉属性 */
  background-color: #f5f5f5;
  border-radius: 4px;
  box-shadow: 0 2px 5px rgba(0,0,0,0.1);

  /* 其他属性 */
  opacity: 1;
  transition: all 0.3s ease;
}

CSS选择器规范

避免使用过于具体的选择器：

不良示例：
div#header ul.nav li a {
  /* 样式 */
}
良好示例：
.nav-link {
  /* 样式 */
}

CSS代码组织

CSS文件结构

建议按功能模块组织CSS文件：

/* 1. 重置和基本样式 */
/* 2. 工具类（Utilities） */
/* 3. 布局（Layout） */
/* 4. 组件（Components） */
/* 5. 页面特定样式（Pages） */
/* 6. 主题（Themes） */
/* 7. 响应式（Responsive） */

使用CSS变量

定义可复用的CSS变量：

:root {
  --primary-color: #3498db;
  --secondary-color: #2c3e50;
  --font-size-base: 16px;
  --spacing-unit: 1rem;
  --border-radius: 4px;
}

.button {
  background-color: var(--primary-color);
  padding: var(--spacing-unit);
  border-radius: var(--border-radius);
}

CSS方法论

使用BEM（Block Element Modifier）命名方法论：

/* Block */
.card { }

/* Element */
.card__header { }
.card__body { }
.card__footer { }

/* Modifier */
.card--highlighted { }
.card--small { }

BEM命名规范

BEM示例：

<div class="search-form">
  <input class="search-form__input">
  <button class="search-form__button search-form__button--disabled">搜索</button>
</div>
.search-form {
  display: flex;
}

.search-form__input {
  flex: 1;
  padding: 0.5rem;
}

.search-form__button {
  padding: 0.5rem 1rem;
  background: blue;
  color: white;
}

.search-form__button--disabled {
  background: gray;
  cursor: not-allowed;
}

响应式设计规范

移动优先

采用移动优先的响应式设计方法：

/* 基础样式（移动设备） */
.container {
  padding: 1rem;
  width: 100%;
}

/* 中等屏幕（平板） */
@media (min-width: 768px) {
  .container {
    padding: 2rem;
    max-width: 720px;
    margin: 0 auto;
  }
}

/* 大屏幕（桌面） */
@media (min-width: 1200px) {
  .container {
    max-width: 1140px;
  }
}

响应式单位

合理使用不同的CSS单位：

/* 固定单位 - 适用于边框等 */
.element {
  border: 1px solid #ccc;
}

/* 相对单位 - 适用于字体大小 */
.element {
  font-size: 1rem; /* 相对于根元素 */
  line-height: 1.5; /* 无单位，相对于字体大小 */
}

/* 视口单位 - 适用于全屏元素 */
.hero-section {
  height: 100vh; /* 视口高度的100% */
  width: 100vw; /* 视口宽度的100% */
}
提示： 使用rem单位设置字体大小，使用em单位设置与字体大小相关的属性，使用%设置容器宽度，使用vw/vh设置与视口相关的尺寸。
